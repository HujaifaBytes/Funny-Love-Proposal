Core Technology Stack

Python

The Foundation

The entire game logic, state management, and data processing is built on Python 3, leveraging its simplicity and powerful data handling capabilities.

Pygame

The Engine

Pygame serves as the core library for rendering graphics, handling user input (keyboard/mouse), managing game state loops, and playing sounds and music.

Openpyxl

The Data Store

This library enables robust data persistence, allowing the game to save session results and high scores directly to a formatted `.xlsx` Excel file.

Dynamic Challenge System

To ensure high replayability, the game's difficulty increases over time through two primary mechanics: linear speed scaling and randomized attack patterns.

Difficulty Scaling Over Time

Every 30 seconds of gameplay, the "Danger Level" rises, which directly increases the base speed of all enemies on screen.

Enemy Attack Behaviors

Enemies dynamically switch behaviors to keep the player engaged and prevent predictable patterns.

Normal Behavior

Enemies move horizontally in formation, descending upon reaching the screen edge.

Dive Attack Behavior

Data Persistence & Analytics

A key project requirement was creating a persistent record of game sessions. All data is saved in a cleanly formatted Excel file for easy analysis.

High Score System

On startup, the application reads the entire Excel log to find and display the all-time high score on the home screen, creating a competitive incentive.

Graceful Failure (`atexit`)

If the application window is closed unexpectedly during gameplay, the `atexit` module ensures the session is still saved with a "Terminated" status, preventing data loss.
